# Welcome to Veyor Marketplace

Veyor is a high-performance marketplace platform designed with a multi-tenant, cloud-native architecture.

## Platform Engineering Layer

This documentation covers the platform engineering layer that manages the full code lifecycle for Veyor services.

### Core Architecture
- **Backend**: Java (Spring Boot) / Go microservices
- **Frontend**: Next.js / TypeScript
- **Database**: RDS PostgreSQL
- **GitOps**: Argo CD
- **Pipelines**: GitHub Actions & Argo Workflows
- **Observability**: Prometheus, Loki, Grafana, OpenTelemetry

### AI Developer Agents
The platform provides **Golden Paths** for AI agents to contribute code, manage infrastructure, and perform automated testing.
Check the [Golden Paths](golden-paths.md) section for integration details.
